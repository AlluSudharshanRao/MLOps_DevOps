# Shared inference contract (joint deliverable)

Training, serving, and data roles must agree on one **input** sample and one **model output** sample per model.

- Replace `sample_inference_request.json` and `sample_model_output.json` with your team’s final agreed payloads before submission.
- The examples here match the **Multi-Tone Assistant** shape from the proposal: draft text in, ranked tone suggestions out.
